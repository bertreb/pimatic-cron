pimatic cron plugin
===================
Provides a clock module, so actions can be triggert at a specific time

Provided predicates
-------------------
For example:

 * the predicate `"Sep 12-13"` is true on the 12th of october, 2013 from 0 to 
   23.59 o'clock 
 * the predicate `"friday 17:00"` is true every friday at 5pm.
